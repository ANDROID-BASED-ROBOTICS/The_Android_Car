The_Android_Car
===============

Android App:
Android_CAR.apk

Android App (Eclipse project):
Android_CAR

Computer Client (Eclipse project):
PC_UDP_client

You will have to start the client running on the PC first, then start the Android app.


Requirements
------------

App:
- Android 2.2
- IOIO libraries

PC client:
- Java Slick 2D library
